📝 Cosa Devi Fare

  1. Leggi latitudine e longitudine dagli input
  2. Fai una GET a https://api.open-meteo.com/v1/forecast con parametri:
    • latitude={lat}
    • longitude={lon}
    • current=temperature_2m,relative_humidity_2m,weather_code
    • timezone=auto
  3. Estrai i dati dal JSON: response.current
  4. Visualizza temperatura, umidità, descrizione meteo
💡 URL Completo: 
https://api.open-meteo.com/v1/forecast?latitude=45.46&longitude=9.19¤t=temperature_2m,relative_humidity_2m,weather_code&timezone=auto
