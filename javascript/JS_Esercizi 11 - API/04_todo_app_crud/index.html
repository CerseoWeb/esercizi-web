📝 Cosa Devi Fare

Questa è l'esercitazione finale! Devi implementare TUTTE le operazioni CRUD:

1️⃣ Carica TODO (GET)

  • Fai una GET a /todos?userId={id} per ottenere i TODO dell'utente
  • Visualizza la lista

2️⃣ Aggiungi TODO (POST)

  • Fai una POST a /todos con: {userId, titolo, completato: false}
  • Ricarica la lista

3️⃣ Modifica TODO (PUT)

  • Fai una PUT a /todos/{id} con: {completato: !currentValue}
  • Aggiorna la lista

4️⃣ Elimina TODO (DELETE)

  • Fai una DELETE a /todos/{id}
  • Rimuovi dalla lista
💡 Struttura del codice: 
// Funzioni che devi implementare:
loadUserTodos(userId)     // GET
addTodo()                 // POST
toggleTodo(id, current)   // PUT
deleteTodo(id)            // DELETE
displayTodos(todos)       // Visualizza (già fatto)
